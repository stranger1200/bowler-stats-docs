Commands

Use these slash commands in Discord after inviting the bot. You can also open the Help menu in-app via /help to browse or search command pages.

Profile & stats

/profile
View your Clash Royale profile (or another player's). Shows trophies, arena, wins/losses, clan, Path of Legends, masteries and card stats. Optional: user or tag.
/battlelog
View your Clash Royale battle log (or another player's). Browse recent battles, switch rounds in duels, and use View Summary for win rates and card performance from your last 25 battles. Optional: user or tag.
/deck
View Clash Royale decks. Subcommands: show – your most used deck (or another player's, optional deck URL); latest – deck from your most recent battle; random – random deck with random tower troop; ranked – random deck from a top Ranked player.
/progress
View your Clash Royale progress (or another player's): card levels, upgrade costs, masteries, available upgrades and misc achievements. Optional: user or tag.

Account & data

/tag
Manage your saved Clash Royale player tags. save – save a tag (optional name, e.g. Main/Mini); remove – remove a saved tag; select – set which account is active; list – list all saved tags. Supporters can save more tags.
/data
Manage your stored data. view – see what the bot stores for your account; delete – permanently delete your saved data; info – information about what data we collect.
/settings
Customise your experience: profile colour (Patreon: custom hex), public profile, private responses (ephemeral), anonymous mode (hide in-game identifiers), display alt text on images, and show bot tips.

Achievements

/achievement
View and manage your achievements. claim – check and claim available achievements; list – view all available achievements; progress – view progress towards unclaimed achievements.

Utility & support

/help
View help information for bot commands. Use the dropdown to switch pages, or use the view option to search for a specific command or page.
/invite
Get an invite link to add the bot to other servers.
/ping
Check the bot's latency and the ping to the Clash Royale API.
/patreon
View information about supporting the bot on Patreon and the perks (e.g. custom colours, more tag slots, lower cooldowns).

Server settings

/friend-links
Manage automatic friend link profile detection for this server (requires Manage Server). toggle – enable or disable; whitelist-channel / blacklist-channel – limit which channels are monitored; status – view current settings.

Context menu

Right‑click a user in Discord and use Apps (or Bowler Stats) to run:

  • View Profile – View that user's Clash Royale profile (uses their saved tag).
  • View Battle Log – View that user's battle log (uses their saved tag).
