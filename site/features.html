Features

Profile & stats

  • View your Clash Royale profile (level, trophies, badges, etc.)
  • Link and save your player tag to your Discord account
  • Share profile embeds in the server

Battle logs

  • Recent battles with deck and result
  • Battle history on demand via slash commands

Battle history

  • See your recent battles and results at a glance
  • Data fetched from the official Supercell API

Data & privacy

  • /data view – See what data the bot stores for your account
  • /data delete – Permanently delete your saved data

Customisation (Patreon)

  • Custom profile colours (hex colour code)
  • Other supporter perks as announced

Server settings

  • Server-specific configuration (e.g. whitelisting)
  • Managed by server admins; data removed when the bot leaves the server

See the Commands page for the full list of slash commands and options, or use /help in Discord.
